Internal consistency check — ultimate analysis sum

C + H + O + N + S + Cl + Ash (d.b.) should sum to ~100%:

S₀
43.1 + 5.27 + 37.3 + 0.61 + 0.04 + 0.09 + 3.9
= 90.3%
~9.7% gap = unlisted fixed carbon (normal)
SW₁
45.7 + 6.02 + 39.0 + 0.32 + 0.03 + 0.06 + 2.2
= 93.3%
~6.7% gap = unlisted fixed carbon (normal)
SW₂
45.4 + 6.08 + 38.9 + 0.34 + 0.03 + 0.08 + 2.1
= 92.9%
~7.1% gap = unlisted fixed carbon (normal)

Volatile matter + fixed carbon + ash = 100% on d.b. basis. The gaps here are entirely expected — fixed carbon was not separately reported in the original study. The dataset is internally consistent.
